# GovHack2016_Where2Go
## Setup
1. Install Node.js
2. Install Android SDK
3. 'npm install -g cordova'
4. 'npm install -g bower'
5. clone down project and open open folder in terminal

### App 
1. 'cd app'
2. 'bower install'
3. 'cordova platform add android'
4. 'cordova run android'

### Server
1. 'cd server'
2. 'bower install'
3. 'npm install'
4. 'node index.js'
